System starten & stoppen

AmbientCT läuft lokal in Docker. Kein Internet nötig nach der Installation.

Starten

Öffnen Sie ein Terminal und tippen Sie:

cd ~/dev/AmbientCT
docker compose up -d

Warten Sie ca. 30 Sekunden. Dann prüfen:

docker compose ps

Beide Container müssen Up und healthy anzeigen.

Panorama-Rekonstruktion (Dental CPR)

Die CPR-Funktion (Curved Planar Reformation) erzeugt ein klassisches Panoramabild aus einer CBCT-Aufnahme. Sie zeichnen einmal den Zahnbogen — der Rest ist automatisch.

Voraussetzung: Eine CBCT-Studie muss geöffnet sein. AmbientCT wechselt automatisch in den Dental-Modus mit Axial-Ansicht links und CPR-Panel rechts.

Ersten Punkt setzen

Im linken Panel (Axial-Ansicht) auf den hinteren Molaren links klicken

Zahnbogen nachzeichnen

8–12 Punkte entlang des Bogens setzen: Molar links → Prämolaren → Frontzähne → Prämolaren → Molar rechts

Kurve abschließen

Doppelklick auf den letzten Punkt (hinterer Molar rechts)

Panorama ansehen

Das rechte Panel zeigt sofort das rekonstruierte Panoramabild

Slab-Dicke anpassen

Slab-Regler (1–20 mm): Dünn (1–3 mm) = Einzelschicht · Dick (5–15 mm) = MIP wie klassisches OPG

Häufige Fragen

Der Viewer lädt nicht
Warten Sie 30–60 Sekunden nach dem Start. Prüfen Sie mit docker compose ps, ob beide Container Up anzeigen. Dann Seite neu laden (F5).
Ich sehe keine Bilder
Prüfen Sie, ob der DICOM-Import abgeschlossen ist. Öffnen Sie http://192.168.29.142:8042 und überprüfen Sie, ob die Studie dort erscheint. Dann Viewer neu laden.
Das Passwort für Orthanc stimmt nicht
Das Passwort steht in der Datei .env im AmbientCT-Ordner (Variable ORTHANC_PASSWORD). Fragen Sie Ihren Administrator.
Nach Neustart des Computers läuft AmbientCT nicht
Docker startet nicht automatisch. Führen Sie nach dem Neustart aus: cd ~/dev/AmbientCT && docker compose up -d
Wie sehe ich Fehler-Logs?
docker compose logs viewer — OHIF Viewer
docker compose logs orthanc — Orthanc PACS
Wie viel Speicherplatz wird benötigt?
Ein CBCT-Datensatz benötigt typisch 50–500 MB. Empfohlen: SSD mit mindestens 100 GB für den Datenspeicher.
