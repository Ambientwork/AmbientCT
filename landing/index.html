Any DICOM Source

Drag & drop files or receive from any CBCT, CT, MRI device.

# Clone, setup, run
git clone https://github.com/Ambientwork/AmbientCT.git
cd AmbientCT
./scripts/setup.sh
docker compose up -d

# Open in browser
open http://localhost:3000
