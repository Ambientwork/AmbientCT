# Herunterladen und starten

cd AmbientCT
./scripts/setup.sh
docker compose up -d

open http://localhost:3000
Funktionen

Alles was Ihre Praxis braucht

🏥

Vollwertiger PACS-Server

Orthanc mit DICOMweb, C-STORE und WADO. Empfangen Sie Bilder direkt von jedem DICOM-Gerät in Ihrem Netzwerk.

🧠

3D-Darstellung

Axiale, sagittale und koronale MPR-Ansichten mit Cornerstone3D. GPU-beschleunigte WebGL-Darstellung direkt im Browser.

🦷

Dental-Presets

Fenster-/Levelwerte optimiert für Knochen, Weichgewebe, Implantate, Mandibularkanal und Atemwegsanalyse.

📦

Ein Befehl genügt

Docker Compose auf Mac, Linux und Windows. Von Null zum laufenden PACS in unter 5 Minuten.

🔒

Datenschutz zuerst

Läuft komplett in Ihrem lokalen Netzwerk. Keine Cloud, kein Tracking, keine Analyse. Patientendaten bleiben bei Ihnen.

📂

Jede DICOM-Quelle

Dateien per Drag & Drop, über die Kommandozeile oder direkt von CBCT-, CT- oder MRT-Scannern via DIMSE C-STORE.

Architektur

Einfach und eigenständig

  Ihr Praxis-Netzwerk

  Browser  ────────>  OHIF Viewer :3000
  (Behandler)           React + Cornerstone3D + WebGL
                             │
                             │ DICOMweb (REST)
                             ▼
  CBCT / Scanner  ──>  Orthanc PACS :8042
  (DIMSE C-STORE)       :4242 (DICOM-Port)
                             │
                             ▼
                      SQLite + Dateisystem
                      Docker Volume (orthanc-db)
