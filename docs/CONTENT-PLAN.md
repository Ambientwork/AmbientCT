# Content & Marketing Plan — AmbientCT + AmbientWP

## Overarching Narrative

**"Ein Zahnarzt baut mit AI-Agenten professionelle Software — live."**

Das ist keine Coding-Tutorial-Serie. Das ist eine Founder-Story:
Ein Non-Programmer nutzt Conductor, Claude Code und gstack um
echte, nutzbare Open-Source-Tools zu bauen. Der Twist: die Tools
lösen reale Probleme in der Branche, in der er arbeitet.

### Warum das funktioniert

1. **Relatability:** Zahnärzte, Ärzte, KMU-Inhaber sehen sich selbst — "wenn der das kann..."
2. **Credibility:** Es sind keine Toy-Apps, sondern ein Medical Imaging Server und ein SaaS-Tool
3. **Virality:** AI + Non-Coder + Medical = drei Aufmerksamkeits-Trigger gleichzeitig
4. **Funnel:** Jeder Content-Piece hat ein natürliches CTA → AmbientCT Download ODER Ambientwork Kontakt

## Red Lines (was wir NICHT tun)

1. **Keine echten Patientendaten** in Screenshots, Videos, Demos — immer anonymisiert oder Sample-Daten
2. **Kein "AI ersetzt Ärzte"** Messaging — AmbientCT ist ein Convenience-Tool, kein Diagnose-Tool
3. **Keine falschen Versprechen** über FDA/CE-Zertifizierung — Disclaimer immer sichtbar
4. **Kein Bashing** von kommerzieller Software (Planmeca, Carestream etc.) — wir positionieren als Ergänzung, nicht Ersatz
5. **Keine AI-Hype-Sprache** ("revolutionär", "disruptiv") — sachlich, ehrlich, authentisch
